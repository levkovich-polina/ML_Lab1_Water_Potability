# ML_Lab1_Water_Potability

ДАТАСЕТ: https://www.kaggle.com/datasets/adityakadiwal/water-potability

1. Описание задачи:
   
	Цель: построить модель машинного обучения, которая на основе физико-химических характеристик воды будет предсказывать, пригодна ли вода для питья. 
	Имеется набор данных, содержащий девять признаков, характеризующих качество воды:

		1. ph (уровень кислотности)
		2. Hardness (жёсткость воды)
		3. Solids (общее содержание растворённых веществ)
		4. Chloramines (хлорамины)
		5. Sulfate (сульфаты)
		6. Conductivity (электропроводность)
		7. Organic_carbon (органический углерод)
		8. Trihalomethanes (тригалометаны)
		9. Turbidity (мутность)
	Также есть целевая переменная Potability принимает значения:
		0 - вода непригодна для питья
		1 - вода пригодна для питья
	Тип задачи: бинарная классификация
	
3. Шаги:
	- Обработка пропущенных значений
	- Нормализация признаков 
	- Учёт несбалансированности классов 
	- Обучение и сравнение моделей
	- Применение SMOTE для балансировки обучающей выборки
	- Оценка качества по F1-score

4. Использовались модели:
	- k-ближайших соседей (k-NN)  
	- Логистическая регрессия  
	- Случайный лес  
	- XGBoost  
Также протестирован метод SMOTE для балансировки классов.

5. Лучший результат:
	- Модель: Случайный лес с SMOTE  
	- F1-score на тестовой выборке: 0.578
	- (Базовая версия без SMOTE: F1 = 0.532)
	
6. Вывод:
	В работе была успешно решена задача классификации воды на пригодную и непригодную для питья.
	Среди протестированных моделей лучшей оказалась Случайный лес, особенно после применения SMOTE для борьбы с дисбалансом классов - F1-score достиг 0.578.
